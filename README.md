# 💪 Мій план тренувань — v2

Персональний PWA-додаток з планом тренувань та харчування.

## Нові фічі v2

- ✅ **Відмітка тренувань** — кнопка «Виконано» + лічильник у шапці
- 🌙 **Темна тема** — перемикач + автовизначення системної теми
- 📴 **Офлайн-режим** — Service Worker кешує всі файли
- 📱 **PWA** — встановлюється на домашній екран iPhone

## Структура проєкту

```
fitness-plan/
├── index.html          # Розмітка
├── manifest.json       # PWA маніфест
├── sw.js               # Service Worker (офлайн)
├── css/
│   └── style.css       # Стилі (light + dark тема)
├── js/
│   ├── data.js         # Дані (тренування, харчування)
│   └── app.js          # Логіка додатку
├── icons/
│   ├── icon-192.png    # Іконка для PWA
│   └── icon-512.png    # Іконка для PWA
├── generate_icons.py   # Скрипт генерації іконок
└── README.md
```

## Як відкрити локально

Відкрий `index.html` у браузері. Service Worker потребує HTTPS або localhost.

Див. покрокову інструкцію нижче.
