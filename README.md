# 💪 Мій план тренувань

Персональний PWA-застосунок з планом тренувань та харчування.

## Функціонал

- 🏋 **Тижневий план тренувань** — акордеон із YouTube-відео, випадковий вибір із 2–3 варіантів
- ⏱ **Таймер відпочинку** — фіксована панель знизу, звук і вібрація при завершенні
- 🥗 **Двотижневий план харчування** — динамічне масштабування порцій під профіль
- 👤 **Профіль** — вага, зріст, вік, стать, рівень активності, ціль
- 🔢 **Підрахунок калорій** — формула Міффліна–Сент Жеора, три цілі (кат/балк/рекомп)
- ✅ **Відстеження тренувань** — відмітка «Виконано» + лічильник за тиждень
- 🌙 **Темна тема** — перемикач + автовизначення системної теми
- 📴 **Офлайн-режим** — Service Worker, cache-first для ресурсів
- 📱 **PWA** — встановлення на домашній екран iPhone через Safari
- 📅 **Експорт до Apple Calendar** — генерація .ics на 12 тижнів
- 🔔 **Push-нагадування** — за 20 хв до тренування (поки вкладка відкрита)

## Структура проєкту

```
Plan_site/
├── index.html              # Розмітка (без inline onclick)
├── manifest.json           # PWA маніфест
├── sw.js                   # Service Worker (кеш + офлайн)
├── css/
│   ├── styles.css          # Точка входу (@import)
│   ├── base/
│   │   ├── variables.css   # CSS-змінні, теми
│   │   ├── reset.css       # Базовий reset
│   │   └── layout.css      # Сітка, header, контент
│   ├── components/         # accordion, cards, forms, modals…
│   └── utils/
│       └── responsive.css  # Медіазапити
├── js/
│   ├── main.js             # Точка входу: init + всі слухачі подій
│   ├── core/
│   │   ├── state.js        # Глобальний стан (єдине джерело правди)
│   │   ├── calc.js         # Формули (без DOM): BMR, TDEE, макроси, порції
│   │   └── storage.js      # localStorage: профіль, відмітки тренувань
│   ├── data/
│   │   ├── profile.js      # Константи: GOALS, ACTIVITY_FACTORS, DEFAULT_PROFILE
│   │   ├── workouts.js     # Розклад тренувань, вправи, дані для .ics
│   │   └── nutrition.js    # План харчування тижень 1 і 2
│   ├── features/
│   │   ├── workout.js      # Рендер тренувань, акордеон, відмітки
│   │   ├── nutrition.js    # Рендер харчування і макро-панелі
│   │   ├── profile.js      # UI профілю: статистика, модалка налаштувань
│   │   └── calendar.js     # Генерація .ics для Apple Calendar
│   ├── utils/
│   │   ├── theme.js        # Темна / світла тема
│   │   └── ui.js           # Toast, офлайн-банер, SW-реєстрація, switchTab
│   ├── timer.js            # Таймер відпочинку між підходами
│   └── notifications.js    # Нагадування про тренування
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## Локальна розробка

Відкрити через Live Server (`127.0.0.1:5501`). Service Worker потребує HTTPS або localhost.

Після змін — жорстке оновлення: **Ctrl+Shift+R** (або DevTools → Application → Service Workers → Skip waiting).

## Деплой

Через **GitHub Desktop** на GitHub Pages. Після деплою піднімаємо `CACHE_VERSION` у `sw.js` — тільки це потрібно для оновлення кешу на всіх пристроях.
